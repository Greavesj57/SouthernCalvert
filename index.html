Streaming Setup/Config

  1. Check PC network setting & ensure online with Ethernet, not WiFi connection
  2. Open Chrome browser & select YouTube.com
  3. Select SCBC icon/channel on upper right
  4. Select Create Video -> Go Live
  5. Select most recent Sunday livestream with 1080p encoding and Click “Reuse Settings”
  6. Change Sermon title on first line & Ensure description is “Sunday Morning Worship Livestream”
  7. Set date/time (10:25) for start of countdown
  8. Check thumbnail to ensure high-resolution
  9. Select “Create"
  10. Scroll down settings to ensure proper encoding and “Low Latency” selected

Wirecast Setup

  1. close untitled window
  2. Open Wirecast, select File->Open_>”Default_SCBC – Sunday”
  3. Select the layer “Decklink mini rec”
  4. Expand left flyout if necessary, and click System Device Properties Ensure audio delay to “0.00” sec. (May adjust later 0.1 – 0.25)
  5. Select Output Menu & Output Settings
  6. Select the Record to Disc output
  7. Change name and file to current date
  8. Ensure Intel Quicksync 1080p30 encoding is selected
  9. Select the YouTube output
  10. Change date in name
  11. Ensure Intel Quicksync 1080p30 encoding is selected
  12. Select “upcoming event” and the scheduled stream previously created
  13. Select the Facebook output
  14. Ensure Intel Quicksync 1080p30 encoding is selected
  15. Enter the sermon title and date
  16. Select “Create” to save the entered values
  17. Select “Optimize Connection” (May repeat or delay until 10:20)
  18. Save Output Settings

Sharing Video

  1. In Chrome browser, select the “Share” option on the upper right in YouTube
  2. Copy the YouTube link from the “Share” dialog
  3. Copy Link to SCBC Public Facebook Page
  4. Copy Link to SCBC Private Group
  5. Alternately the livestream can always be accessed at: www.youtube.com/c/southerncalvertbaptistchurch/live

Wirecast Audio Setup

  1. In Wirecast, select Window->Audio Mixer
  2. On output side of window, select Settings (gear icon) by Noise Reduction Ensure Noise Reduction set to “0” for band, then adjust to ~ 25% for sermon
  3. Prior to countdown, check size and positioning of sermon lower-thirds. Typically sized up a couple of notches & shifted down slightly

Countdown Start

  1. Ensure keying off on video mixer, and channel 5 is selected output PGM
  2. Ensure computer volume is on at proper level in mix 7/8
  3. In Wirecast, start recording and streaming
  4. n YouTube ensure stream is received and select “Go Live”
  5. On another device (not streaming PC, and not using wifi) Monitor audio levels and mix.
    • Matrix 1-2 fed to video room (Ipad custom 2)
    • Mix 5-6 for band (Ipad custom 3)
    • Mix 7-8 for Speaking/house/computer (Ipad custom 1)
  6. Check Facebook stream to ensure up & running
  7. Check both FB & YouTube streams for audio latency (Adjusting latency will affect all outputs, so only change if both streams are offline)

Best Practices

  • Practice dynamic use of manual camera PTZ so it can be done smoothly.Prefer dynamic use of cameras during worship set
  • Make notes during practice of key shots to use at beginning/end of songs & transitions to pulpit for announcements, prayer, sermon & closing. Recommend camera 1 as pastor approaches pulpit & band clears stage, Be prepared to quickly switch to camera 2, scene 1 before titles show
  • Try not to change camera shot during prayer or scripture reading
  • Try to avoid congregational views during health emergency restrictions
  • Try to avoid band chatter & motion during prayer/scripture reading
  • Try to keep pastor centered in shot (center third for cameras 1 & 3)
  • Try to keep same margin above pastor’s head on all camera angles, noting there is less margin at top of frame in Wirecast
  • With experience, can turn keying off between points to use camera fades vs. cuts.Follow sermon outline to be aware of when points will be displayed (keying on)
  • Monitor uplink quality & if it turns red, check upload speed at: www.testmy.net/upload
  • If poor upload speed & dropped streams confirmed, capture screen/notes to pass along to Comcast
